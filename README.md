# ToDoList - DevClub

Aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro, inicialmente baseada em um vídeo do DevClub.  
Após acompanhar o conteúdo, fiz ajustes visuais, reorganizei a estrutura e refatorei partes da lógica para melhorar o funcionamento e a organização do código.

## Sobre o projeto

Este projeto consiste em uma aplicação simples de gerenciamento de tarefas, onde o usuário pode:

- Adicionar novas tarefas
- Remover tarefas individualmente
- Visualizar a numeração dinâmica das tarefas
- Marcar tarefas como concluídas
- Inserir textos longos sem quebrar o layout

O foco principal foi praticar manipulação de DOM, organização de layout com Flexbox e controle de fluxo em JavaScript.

## Melhorias implementadas

Além do que foi apresentado no vídeo original, implementei:

- Sistema de numeração dinâmica das tarefas
- Atualização automática da numeração após exclusão
- Ajustes para evitar quebra do layout com textos muito longos
- Controle do comportamento do Flexbox para impedir distorção do checkbox
- Melhorias visuais com transições e efeitos hover
- Organização mais clara das classes CSS
